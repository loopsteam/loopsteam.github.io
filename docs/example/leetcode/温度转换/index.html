    温度转换 #

    问题描述 #

    给定一个四舍五入到两位小数的非负浮点数 celsius（摄氏度），将其转换为开氏度（Kelvin）和华氏度（Fahrenheit），并以数组 ans = [kelvin, fahrenheit] 的形式返回结果。与实际答案误差不超过 10^-5 的答案视为正确。

    转换公式：

    • 开氏度 = 摄氏度 + 273.15
    • 华氏度 = 摄氏度 × 1.80 + 32.00

    示例 #

    示例 1 #

    输入：celsius = 36.50
    输出：[309.65000, 97.70000]
    解释：

    • 开氏度：36.50 + 273.15 = 309.65
    • 华氏度：36.50 × 1.80 + 32.00 = 97.70

    示例 2 #

    输入：celsius = 122.11
    输出：[395.26000, 251.79800]
    解释：

    • 开氏度：122.11 + 273.15 = 395.26
    • 华氏度：122.11 × 1.80 + 32.00 = 251.798

    提示 #

    • 0 <= celsius <= 1000

    解题 #

    class Solution(object):
    def convertTemperature(self, celsius):
        kelvin = celsius + 273.15
        fahrenheit = celsius*1.80 + 32.00
        return [kelvin,fahrenheit]
